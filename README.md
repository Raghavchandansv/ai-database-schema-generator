# AI Database Schema Generator

## 🚀 Overview
This project converts business requirements into database schemas, SQL queries, and visual diagrams.

## 💡 Features
- Natural language input
- Schema generation (tables & relationships)
- SQL query generation
- Visual schema diagram
- Dynamic response system

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript

## ▶️ How to Run
1. Install dependencies:
   pip install flask flask-cors

2. Run the server:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000/

## 📌 Example
Input:
"student management system"

Output:
- Tables
- Relationships
- SQL queries
- Visual diagram
